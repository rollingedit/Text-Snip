# OCR Snip

Press `Win+Shift+O`, drag over text, and release. OCR Snip copies recognized text to your clipboard.

The installer creates a Start Menu shortcut and selects the desktop shortcut option by default. When OCR Snip is open, pin it from its taskbar icon if you want one-click access.

## Features

- Local OCR
- No cloud OCR
- No screenshots saved
- CPU-only ONNX Runtime inference
- Windows x64 tray app
- Bundled PP-OCRv6 small detector and recognizer models

## Settings

- Hotkey
- Launch at login
- Memory mode
- Small text boost
- Toasts
- Copy mode

## Troubleshooting

- If the hotkey is already in use, OCR Snip opens settings.
- If the clipboard is busy, OCR Snip opens the recognized text in a small result window.
- Protected video or privileged windows may capture as black and produce no readable text.
- OCR models are bundled and verified with SHA-256 hashes; the app does not download models at runtime.
